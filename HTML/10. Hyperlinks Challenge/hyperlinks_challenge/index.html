My Experience

The module is about not just learning the languages that allows an individual to write their own web page but an understanding of what a web page is. From what I understanding, development is not as simple as learning languages and practising mindlessly but to learn the underlying concepts of why a piece of code does what it does. This means as a programmer, it is encouraged to develop an inquisitive mind, asking why when thinking about anything.

I have experience programming in Java and it has been advantageuous in the regard that I am able to understand the concepts quicker but HTML is a totally different language in comparisson. It has been a new experience learning a totally new language.

In regards to the other aspects of the course, Professional Development is an interesting and unique module. From what I have experienced, I assume it is a module that trains the skills needed in the work place, other than just being able to program. This will allow a person to branch off into a different field other than programming within development.
